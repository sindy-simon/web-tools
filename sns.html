SNS文字数チェッカー

投稿文を貼ると、各SNSの文字数上限に収まるかを一括表示。Xは全角2・半角1・URL23の正しい数え方で計算します。

SNS使用上限残り

Xのみ「全角=2・半角=1・URL=一律23」の重み付き、その他は文字数(コードポイント)で計算。入力内容は端末内だけで処理されます。上限値は 2026-06 時点の一般的な公開値で、各SNSの仕様変更で変わることがあります。

SNSごとの文字数の数え方

  • X(旧Twitter): 日本語などの全角は2文字、半角英数記号は1文字、URLは長さに関わらず23文字、絵文字は2文字として計算されます(通常投稿の上限280=全角なら約140文字)。
  • Instagram・TikTok・Threads など: 全角・半角を区別せず「文字数」で数えます(日本語1文字=1)。
  • 海外製のカウンターは英語前提で全角や絵文字の数え方が異なることが多いため、日本語投稿では実際の上限とズレることがあります。

関連ツール: 文字数カウント ／ 全角⇔半角変換

SNS文字数チェッカーについて

SNS はサービスごとに投稿できる文字数の上限が異なり、しかも文字の数え方も同じではありません。特に X(旧Twitter)は、日本語などの全角文字を2、半角英数字を1、URL を長さに関わらず一律23として計算する独自方式です。海外製のカウンターは英語前提のことが多く、日本語の投稿では実際の上限とズレがちです。このツールは投稿文を貼るだけで、主要 SNS の上限に収まるかを日本語の数え方に合わせて一括でチェックします。

こんなときに便利

  • 同じ文章を複数の SNS に同時投稿したいとき、どこで切れるかを事前に確認。
  • X で「あと何文字書けるか」を全角2の正しい数え方で把握。
  • プロフィール文やキャプションが上限内に収まるかのチェック。

よくある質問

X の上限は280なのに、なぜ日本語は約140文字なのですか？

X は全角1文字を2としてカウントするため、全角だけなら 280 ÷ 2 = 約140文字が目安になります。半角英数字や URL を混ぜると、その分もう少し多く書けます。

表示される上限値は常に最新ですか？

表示している上限は一般的な公開値です。各 SNS は仕様を変更することがあるため、重要な投稿の前には公式の案内もあわせてご確認ください。

投稿文は送信されますか？

されません。文字数の計算はすべてブラウザ内で行われ、入力した投稿文が外部に送信されることはありません。
