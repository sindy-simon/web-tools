JSON整形・検証

JSONを貼ると整形して妥当性をチェック。AIが生成したJSONが正しいかの確認にも使えます。

ブラウザ標準の JSON 解析で処理します。入力内容は端末内だけで処理され、サーバーには送信されません。

使い方・補足

  • 整形すると入れ子構造が見やすくなり、どこにエラーがあるかも分かります。
  • 不正なJSONの場合はエラー内容を表示します(末尾カンマ・シングルクォート・キーの引用漏れなどが原因)。
  • 「圧縮(1行)」は余分な空白を除いてサイズを小さくします。

関連ツール: 文字数カウント

JSON整形・検証ツールについて

JSON(JavaScript Object Notation)は、設定ファイルや API のやり取りで広く使われるデータ形式です。人が読み書きしやすい反面、カンマや波カッコ、引用符が一つでも欠けると全体が読み込めなくなります。このツールは、JSON を貼り付けるだけで自動的にインデントを整え、構造を見やすく整形します。同時に文法の妥当性をチェックし、エラーがあればその内容を表示します。AI が生成した JSON が正しい形式かを確かめるのにも便利です。

主な機能

  • 2スペース・4スペース・タブから選べるインデント整形。
  • 余分な空白を除いてサイズを小さくする「圧縮(1行)」。
  • 末尾カンマ・シングルクォート・キーの引用符漏れなど、よくある誤りの検出。

よくある質問

どんな JSON エラーがよくありますか？

多いのは、配列やオブジェクトの末尾に余分なカンマが残るケース、文字列をシングルクォート(')で囲んでいるケース(JSON はダブルクォート"のみ)、キーを引用符で囲み忘れているケースです。エラーメッセージの位置を手がかりに修正してください。

大きな JSON でも処理できますか？

はい。ブラウザ標準の JSON 解析を使っているため高速です。ただし非常に巨大なデータの場合は端末の性能に依存します。

貼り付けた JSON は送信されますか？

されません。整形・検証はすべてブラウザ内で行われ、データが外部サーバーに送信されることはありません。API キーや個人データを含む JSON でも安心して整形できます。
