テキストケースとは

プログラミングでは、複数の単語をつなげて名前を付けるときに、単語の区切り方をルール化した「ケース(命名規則)」を使います。たとえば helloWorld のように2語目以降の頭を大文字にするのが camelCase、hello_world のようにアンダースコアでつなぐのが snake_case です。言語やフレームワークごとに推奨される形式が異なるため、変換が必要になる場面は少なくありません。このツールは、入力した文字列を主要な6形式へ一括で変換します。

主なケースと用途

  • camelCase … JavaScript の変数・関数名。
  • PascalCase … クラス名・コンポーネント名。
  • snake_case … Python の変数・DB のカラム名。
  • CONSTANT_CASE … 定数・環境変数名。
  • kebab-case … URL スラッグ・CSS クラス名。
  • dot.case … 設定ファイルのキー・ログの項目名。

よくある質問

入力はどの形式でもよいですか？

はい。スペース区切り・camelCase・snake_case・kebab-case・PascalCase などを自動認識して単語に分解し、6形式すべてに変換します。

日本語は変換できますか？

ケース変換は英数字を前提としています。日本語や記号が混在すると正しく分割できないことがあるため、英数字の文字列でのご利用をおすすめします。

入力した文字列は送信されますか？

されません。ケース変換はお使いのブラウザ内で行われ、入力した文字列が外部へ送られることはありません。
